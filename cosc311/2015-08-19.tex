\section*{2.1}

Consider these declarative sentences

\[
\begin{cases}
  $``5 is an odd integer''$ \\
  $``6 is an odd integer''$ \\
  $``x $\ge$ 7''$
\end{cases}
\]

Each of these can be converted into a mathematical statement/
predicate, because they can be either true or false.
